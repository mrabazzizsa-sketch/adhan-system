# 🕌 نظام الأذان الآلي

نظام متكامل لإدارة وتشغيل الأذان تلقائيًا في مسجد الشركة، مبني على Python Flask مع واجهة عربية كاملة.

---

## ✨ المميزات

- 🕐 **مواقيت تلقائية** من API أم القرى (aladhan.com) لجميع المدن السعودية
- 🔊 **تشغيل تلقائي** للأذان والإقامة عند دخول الوقت
- 📁 **رفع ملفات صوتية** مختلفة لكل صلاة (أذان + إقامة)
- ⏰ **تعديل المواقيت** يدويًا (زيادة أو نقصان بالدقائق)
- 🔔 **إشعار** قبل الأذان بعدد دقائق قابل للتخصيص
- 📋 **سجل كامل** لجميع أحداث النظام
- 💾 **نسخ احتياطي** للإعدادات
- 📱 **واجهة متجاوبة** تعمل على الجوال والكمبيوتر

---

## 🚀 التشغيل السريع

### Windows
```
انقر مرتين على: start.bat
```
سيتم تثبيت المتطلبات وفتح المتصفح تلقائيًا.

### Linux / Mac
```bash
chmod +x start.sh
./start.sh
```

### يدويًا
```bash
pip install -r requirements.txt
python app.py
```
ثم افتح المتصفح على: **http://localhost:5000**

## ⚙️ الإعداد الأولي

1. **شغّل النظام** باستخدام start.bat أو start.sh
2. افتح **http://localhost:5000** في المتصفح
3. اذهب إلى **الإعدادات** واختر مدينتك
4. **ارفع ملفات الأذان** لكل صلاة (MP3 أو WAV)
5. اضبط **وقت الإقامة** لكل صلاة
6. تأكد من أن **النظام يعمل** (الزر الأخضر في الأعلى)

---

## 🔊 إعداد الصوت

### رفع ملفات الأذان
1. اذهب إلى **الإعدادات → ملفات الأذان والإقامة**
2. اضغط **رفع ملف** بجانب كل صلاة
3. اختر ملف MP3 أو M4A أو WAV
4. اضغط **▶ تشغيل** للتأكد من عمل الملف

### اختبار الأذان
- من لوحة التحكم الرئيسية، اضغط زر **▶** بجانب أي صلاة لاختبار تشغيل الأذان

## 🌐 الاستخدام على الشبكة المحلية

لتشغيل النظام وجعله متاحًا على الشبكة المحلية:

1. **ابحث عن IP جهازك:**
   - Windows: `ipconfig` في CMD
   - Linux/Mac: `ifconfig` أو `ip addr`

2. **شغّل الخادم** (يعمل افتراضيًا على جميع الشبكات: `0.0.0.0`)

3. **افتح من أي جهاز في الشبكة:**
   ```
   http://192.168.x.x:5000
   ```

## 🔧 تشغيل كـ Windows Service (اختياري)

لتشغيل النظام تلقائيًا مع بدء تشغيل Windows:

```bash
pip install pywin32
python -c "import win32serviceutil; ..."
```

أو استخدم **Task Scheduler** في Windows:
1. افتح "Task Scheduler"
2. Create Basic Task → At startup
3. Action: Start program → `python.exe`
4. Arguments: `C:\path\to\adhan-system\app.py`

## 📡 واجهة برمجية (API)

| المسار | الوصف |
|--------|-------|
| `GET /api/prayer-times` | مواقيت الصلاة الحالية |
| `POST /api/upload-audio` | رفع ملف صوتي |
| `POST /api/toggle-system` | تشغيل/إيقاف النظام |
| `POST /api/save-settings` | حفظ الإعدادات |
| `POST /api/save-adjustments` | حفظ تعديلات المواقيت |
| `GET /api/backup` | تنزيل نسخة احتياطية |
| `GET /api/logs` | جلب السجلات |
| `POST /api/refresh-times` | تحديث المواقيت |

## 🛠️ استكشاف الأخطاء

**لا تظهر مواقيت الصلاة؟**
- تأكد من وجود اتصال بالإنترنت
- اضغط "تحديث المواقيت" في لوحة التحكم

**الصوت لا يعمل؟**
- تأكد من رفع ملف صوتي للصلاة
- تأكد من أن مستوى الصوت ليس صفرًا
- تأكد من أن المتصفح مفتوح على الصفحة (للتشغيل من المتصفح)

**النظام لا يشغل الأذان تلقائيًا عند إغلاق المتصفح؟**
- التشغيل التلقائي من الخادم يتطلب تشغيل تطبيق خارجي للصوت
- الحل الأبسط: **إبقاء المتصفح مفتوحًا على الجهاز المتصل بالسماعات**
- أو استخدام مكتبة `playsound` (مذكورة في ملف اختياري)

## 📝 ملاحظات تقنية

- قاعدة البيانات: SQLite في مجلد `instance/`
- مواقيت الصلاة: API من [aladhan.com](https://aladhan.com) — طريقة أم القرى (method=4)
- الجدولة: خيط (Thread) خلفي يفحص الوقت كل 30 ثانية
- التخزين المؤقت: المواقيت تُحفظ في قاعدة البيانات وتُجلب مرة واحدة يوميًا
